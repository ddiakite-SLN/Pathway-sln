# Pathways by SLN 🎓

AI-powered college guidance for first-generation, low-income, and immigrant high school students.

**Built by:** Doussouba Diakite — RITA Tech & Data Intern, Student Leadership Network

---

## Live App
🔗 https://pathway-sln-aolubdmndaygqdagqz65zf.streamlit.app/

## What It Does
- 🏫 Matches students to colleges based on GPA, test scores, financial need, and preferences
- 🎯 Career matching across 176 occupations using BLS salary data
- 💰 Calculates Pell Grant, TAP, Dream Act, and HEOP eligibility
- 📋 Builds a Safety / Match / Reach college list with PDF export

## Data
- **3,690 US colleges** — IPEDS 2023-24 (U.S. Dept. of Education)
- **176 careers** — BLS Occupational Employment Statistics 2023
- **Aid thresholds** — Federal + NY State 2025-26

## Stack
Python · Streamlit · Pandas · ReportLab
